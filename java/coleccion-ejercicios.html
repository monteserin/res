Ejercicios variables

  1. Declarar una variable 'i' de tipo int, una variable 'd' de tipo double y una variable 'c' de tipo char. Asignar a cada una un valor. Mostrar por pantalla valor de cada variable, la suma de 'i' + 'd', y la diferencia de 'i' - 'd'.
  2. Declarar una variable entera 'i' y asígnale un valor. Incrementar 'i' en 77, decrementarla en 3 y duplicar su valor.
  3. Intercambiar el contenido de dos variables
  4. Calcular el número de cifras de un número entero

Leer del teclado

  1. Leer dos números enteros por teclado y mostrarlos por pantalla.
  2. Leer un nombre y mostrar por pantalla: “Buenos dias nombre_introducido”
  3. Leer un número entero por teclado y mostrar por pantalla el doble y el triple.
  4. Leer una cantidad de grados centígrados y pasarlos a grados Fahrenheit. La fórmula correspondiente es: F = 32 + ( 9 * C / 5). El proceso de leer grados centígrados se debe repetir mientras que se responda ‘S’ a la pregunta: Repetir proceso? (S/N)

Funciones

Resolveremos los siguientes ejercicios definiendo una función con parámetros de entrada y que siempre devolverá un valor que procesaremos en la clase main,

  1. Leer por teclado el valor del radio de una circunferencia y mostrar por pantalla la longitud y el área de la circunferencia. Longitud de la circunferencia = 2*PI*Radio, Area de la circunferencia = PI*Radio^2
  2. Programa que pase una velocidad en Km/h a m/s. La velocidad se lee por teclado.
  3. Programa lea la longitud de los catetos de un triángulo rectángulo y calcule la longitud de la hipotenusa según el teorema de Pitágoras.
  4. Programa que calcula el volumen de una esfera.
  5. Programa que calcula el área de un triángulo a partir de la longitud de sus lados.
  6. Programa que lee un número de 3 cifras y muestra sus cifras por separado.

Arrays

  3. Leer 10 enteros y mostrar la media de los valores negativos y la de los positivos.
  4. Programa que lea una variable entera mes, compruebe si el número introducido está entre 1 y 12 y muestre el nombre del mes.
